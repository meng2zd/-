# 集美大学自动评教脚本

Tampermonkey 脚本，自动填写集美大学课堂教学学生评价表。

## 功能

- 第 1-6、8-10 题：自动选 10 分
- 第 7 题：自动选 5 分
- 第 11 题：自动选"否"
- 第 12 题：留空

## 安装

1. 浏览器安装 [Tampermonkey](https://www.tampermonkey.net/) 插件
2. 或手动复制 `jmu-evaluation.user.js` 到 Tampermonkey 新建脚本中粘贴
